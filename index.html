Mission

Games built
from scratch.

The Grid is a personal arcade — a collection of browser-based games designed and built entirely from the ground up. No engines, no templates. Just code, creativity, and a passion for making things that feel good to play.

Each game is crafted with attention to detail — from mechanics to interface. New titles are always in development. The grid keeps growing.

Release Notes
The Grid — Changelog
03
v1.0 DEEP Mar 2026
DEEP Goes Live
A dark, atmospheric reimagining of Minesweeper set on the ocean floor. Scan for depth charges, drop beacon markers, and race the clock — or the tide.
  • Launch & Setup
  • Two modes — Classic (race your best time) and Pressure (rising tide auto-reveals cells from below)
  • Preset + custom difficulty — Easy / Medium / Hard presets with live row, column, and mine sliders
  • Best time tracking — saved locally per exact config (mode + grid size + mine count)
  • Gameplay
  • Flood reveal — clicking an empty cell cascades to all connected safe cells instantly
  • First-click safety — mines never spawn on or adjacent to your first cell
  • Pause / Resume — freezes timer and tide pressure mid-game with a frosted overlay
  • Right-click / long-press to drop amber beacon flags on suspected charges
  • UI & Polish
  • Deep navy + bioluminescent cyan sonar aesthetic — scanline background, pressure gauge timer
  • Staggered depth charge reveal animation on loss, bioluminescent flood animation on reveal
  • Score submitted to global leaderboard (win time in seconds) via Supabase
02
v2.0 Cosmic Blackjack Mar 2026
VOID21 Major Update
The biggest update to Cosmic Blackjack yet. New game modes, multiplayer overhaul, cosmic chaos mechanics, and a full insurance system.
  • Launch & Setup
  • New launch flow — Solo vs Multiplayer split screen replaces old player count picker
  • Multiplayer expanded to 10 players (up from 8), named players + avatar icons per player
  • Bet limits — set a min and max bet at setup, enforced throughout the session
  • Game Modes
  • Free Play — infinite mode, END button only exit
  • Goal Mode — race to a chip target; game ends the round someone hits it
  • Survival Mode — hit 0 chips and you're eliminated; last player standing wins
  • Challenge Mode — hit a chip target within a set round limit or the run ends
  • New Mechanics
  • Insurance — offered when dealer shows an Ace; pays 2:1 on dealer blackjack
  • Void Events — 8 random cosmic effects per round, toggleable with adjustable trigger chance
  • Streak tracking — 🔥 HOT and 🧊 COLD streaks shown per player
  • UI & Polish
  • Inline mode descriptions and void event drawer on setup screen
  • Challenge and Goal progress bars visible during play
  • End game screen adapts title and summary by game mode
01
v1.0 The Grid Jan 2026
The Grid Goes Live
Initial launch of the portal and the first two games — Realm Chess and Cosmic Blackjack.
  • Realm Chess — fantasy chess vs AI or 2P pass-and-play, 3 AI difficulties, full chess rules
  • Cosmic Blackjack (VOID21) — 1–8 players, cosmic card suits, hit/stand/double/split
  • Global leaderboard via Supabase pulling scores from both games
  • Global session counter tracking total visits
  • Now Playing pill — live indicator when a game is being played
  • Portal: comet cursor, 3D card tilt, shimmer, dark/light theme, splash screen
